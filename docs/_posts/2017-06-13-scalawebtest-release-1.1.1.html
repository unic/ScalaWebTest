Highlights of this release

Update from ScalaWebTest <= 1.1.0

No action is required when upgrading from ScalaWebTest 1.1.0.

Fixed a bug in the misfit relevance calculation

We found an error in the calculation of the misfit relevance, see issue 38 for details. Although this didn't effect whether a HtmlPage or HtmlElement fitted the provided gauge, it sometimes led to the wrong reason being shown, in case it didn't fit the gauge. This might be very misleading, therefore an upgrade to version 1.1.0 is highly recommended.
